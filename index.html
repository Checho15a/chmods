Comandos

💡 Se recomienda usar /chpreview para gestionar proyecciones. La GUI es más intuitiva, permite almacenar medios con nombre, previsualizar antes de enviar y ofrece opciones exclusivas no disponibles por comandos.
Comandos generales
/chimage /chvideo /chgif
Muestra los créditos del mod.
/chimage <>
Acceso a los comandos de la sección de imágenes.
/chvideo <>
Acceso a los comandos de la sección de videos.
/chgif <>
Acceso a los comandos de la sección de GIFs.
/chpreview
Abre la interfaz gráfica (GUI) del administrador del mod.
/chimage — Imágenes
/chimage help
Muestra la ayuda de uso de comandos para la proyección de imágenes.
/chimage link "url" true/false
true proyecta la imagen del link especificado. false detiene su proyección.
/chimage off
Desactiva la imagen que se esté proyectando en ese momento. Uso recomendado sobre el comando completo con false.
/chimage on
Vuelve a proyectar la última imagen mostrada, incluyendo los parámetros que tenía configurados.
⚠️ Los links deben terminar en .jpg. Otros formatos como .png o .webp pueden funcionar pero no están garantizados.
Parámetros opcionales de /chimage link
Los parámetros se añaden al final del comando, en cualquier combinación. Por defecto todos están en NO, por lo que no es necesario especificarlos si no se quieren activar.
Parámetro
Función
chatSI
Activa la restricción de chat para jugadores no operadores.
cmdsSI
Activa la restricción de comandos para jugadores no operadores.
interactSI
Bloquea el uso de click derecho e izquierdo para jugadores no operadores.
invSI
Bloquea el acceso al inventario para jugadores no operadores.
moverSI
Bloquea el movimiento de jugadores no operadores durante la proyección.
Ejemplos de uso con parámetros:
/chimage link "url" true chatSI
1 parámetro activo.
/chimage link "url" true chatSI cmdsSI
2 parámetros activos.
/chimage link "url" true chatSI cmdsSI interactSI invSI moverSI
Todos los parámetros activos.
/chvideo — Videos
/chvideo help
Muestra la ayuda de uso de comandos para la proyección de videos.
/chvideo link "url" true/false
true proyecta el video del link. false detiene su proyección. El video se reproduce una sola vez.
/chvideo off
Detiene el video que se esté proyectando.
/chvideo on
Vuelve a proyectar el último video mostrado.
⚠️ El link debe terminar obligatoriamente en .gif. Si se usa un GIF con este comando, se reproducirá una sola vez como si fuera un video, sin repetirse. Los videos no reproducen sonido en esta versión.
/chgif — GIFs
/chgif help
Muestra la ayuda de uso de comandos para la proyección de GIFs.
/chgif link "url" true/false
true proyecta el GIF. false lo detiene. Se reproduce en bucle hasta desactivarse.
/chgif link "url" true 30
Proyecta el GIF con 30% de opacidad. El número es el porcentaje (0–100). Parámetro opcional — sin él, se usa opacidad completa.
/chgif off
Detiene el GIF que se esté proyectando.
/chgif on
Vuelve a proyectar el último GIF mostrado.
ℹ️ El parámetro de opacidad es completamente opcional. Si no se especifica, el GIF se proyecta con su opacidad original al 100%.
Distribución del mod

Descarga

Los mods del proyecto CH son privados y no están disponibles para descarga pública directa. Cada mod es desarrollado de forma independiente y su distribución está controlada para garantizar la integridad del trabajo.
